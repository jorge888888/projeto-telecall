2FA

Two-Factor Authentication
(Autenticação de dois fatores)

O que é?

O 2FA é um procedimento de segurança que garante que serão necessários 2 fatores únicos para liberação de uma ação.
O primeiro fator é a senha que o usuário e o segundo pode ser autenticado via token, via detecção de impressão digital, reconhecimento facial, código enviado via sms, entre outros.

    O 2FA permite que você:

  • Envia a senha via SMS, voz ou e-mail para autenticação do usuário.
  • Adicione uma camada extra de segurança além da senha pessoal.
  • Ofereça maior segurança para usuários.

Benefícios

  • Ofereça segurança aprimorada para seus clientes.
  • API simples e de rápida implementação.
  • Envio de OTP por meio de vários canais, incluindo SMS, voz ou e-mail.
  • Flexibilidade de canais garante que o usuário conseguirá completar a tarefa desejada mesmo quando tiver problema com um dos canais. Exemplo: Enviar OTP por SMS, em caso de falha, enviar OTP por chamada de voz, em caso de outra falha, enviar por e-mail.
  • Reduza casos de fraude e invasões e evite o acesso de bots por invasores.
  • Plataforma intuitiva que permite visualizar relatórios de uso por dia, mês ou ano e pesquisar usando diversos critérios de filtro.
Como Funciona?

Usuário acessa seu site ou aplicativo e digita a senha cadastrada para entrar em seu perfil ou completar uma transação.

A Telecall recebe a tentativa de login e solicita que o usuário insira seu número de telefone para autorizar o acesso.

Após inserir seu número, a Telecall envia para o usuário por SMS, chamada ou e-mail um código PIN de uso único. Esse código PIN é também conhecido como OTP (One Time Password).

O usuário insere o código no site ou aplicativo para concluir o processo de verificação.

Por que usar?

  • Oferece segurança aprimorada para seus clientes.
  • API simples e de rápida implementação.
  • Envio de OTP por meio de vários canais, incluindo SMS, voz ou e-mail.

  • Plataforma intuitiva que permite visualizar relatórios de uso por dia, mês ou ano e pesquisar usando diversos critérios de filtro.
